## 🔧 高级用法

### 批量生成所有页面

创建任务列表文件：
```bash
cat > tasks.txt << 'EOF'
实现登录页面 (Login.vue)
实现用户管理页面 (UserManagement.vue)
实现房源管理页面 (PropertyManagement.vue)
实现财务报表页面 (FinancialReport.vue)
实现客户管理页面 (CustomerManagement.vue)
EOF

# 批量执行
while IFS= read -r task; do
    filename=$(echo "$task" | grep -oP '\(\K[^)]+')
    echo "生成: $filename"
    ./scripts/ask-claude.sh "$task" > "frontend/src/views/$filename"
    sleep 2  # 避免 API 限流
done < tasks.txt
```

## 🎯 最佳实践

### 1. 明确的提示词

❌ **不好的提示词**：
```
生成一个首页
```

✅ **好的提示词**：
```
请实现 Vue 3 首页组件 (HomePage.vue)，使用 Composition API + TypeScript + Element Plus。

功能需求：
1. 顶部蓝色渐变背景区域
   - 左上角：酒店名称下拉选择器
   - 中间：营业额和可售房间数统计
   - 右上角：客服和添加按钮

2. 房态统计横条（5个状态的数量显示）
   - 预抵：0
   - 已抵：0
   - 预离：0
   - 已离：0
   - 新办：0

3. 功能模块网格（3列9行）
   - 每个模块：图标 + 文字
   - 点击跳转到对应页面

4. 底部导航栏（5个Tab）

样式要求：
- 参考附图的蓝色渐变主题
- 响应式布局
- 使用 Element Plus 的组件

代码要求：
- 完整的 TypeScript 类型定义
- 包含模拟数据
- 添加必要的注释
```

## 🐛 常见问题

### 问题 1：API 调用失败

**现象**：
```
Error: API request failed
```

**解决**：
```bash
# 检查 API 密钥
echo $ANTHROPIC_API_KEY

# 检查网络
curl -I https://api.anthropic.com

# 设置代理（如果需要）
export HTTP_PROXY=http://127.0.0.1:7890
export HTTPS_PROXY=http://127.0.0.1:7890
```

### 问题 2：生成的代码不完整

**原因**：输出被截断（max_tokens 限制）

**解决**：分解任务
```bash
# 不要一次生成整个页面，而是分步骤
./scripts/ask-claude.sh "生成 HomePage 的模板部分（template）"
./scripts/ask-claude.sh "生成 HomePage 的脚本部分（script setup）"
./scripts/ask-claude.sh "生成 HomePage 的样式部分（scoped style）"
```

### 问题 3：图片无法识别

**原因**：图片路径错误或格式不支持

**解决**：
```bash
# 检查图片
file ~/.openclaw/workspace/smart-pms/design/ui-reference.jpg

# 如果是 WebP，转换为 JPEG
convert ui-reference.webp ui-reference.jpg

# 压缩大图片
convert ui-reference.jpg -quality 85 -resize 1920x ui-reference-compressed.jpg
```
