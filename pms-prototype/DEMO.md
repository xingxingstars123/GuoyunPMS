## 📊 功能1: 数据可视化仪表盘

### 演示场景
查看民宿最近7天的营收趋势、房型收入占比和各渠道订单分布。

### 演示步骤

#### 1. 访问可视化API
```bash
# 营收趋势(近7天)
curl -s http://localhost:3100/api/dashboard/revenue-trend?days=7 | jq

# 房型统计
curl -s http://localhost:3100/api/dashboard/room-type-stats | jq

# 渠道分布
curl -s http://localhost:3100/api/dashboard/channel-distribution | jq
```

#### 2. 前端展示效果
打开 `/pages/index/index-with-charts.vue` 页面:
- **营收趋势折线图**: 展示每日营收变化,峰值和低谷一目了然
- **房型占比饼图**: 显示大床房、标准间等各房型的营收贡献百分比
- **渠道订单柱状图**: 对比携程、美团、去哪儿等渠道的订单量

#### 3. 实际数据示例

## 🤖 功能2: 智能房间推荐

### 演示场景
客户预订2026年4月20-22日(2晚),预算500元/晚,偏好高楼层,系统智能推荐最合适的房间。

### 演示步骤

#### 1. 调用推荐API
```bash
curl -X POST http://localhost:3100/api/rooms/recommend \
  -H "Content-Type: application/json" \
  -d '{
    "checkIn": "2026-04-20",
    "checkOut": "2026-04-22",
    "maxPrice": 500,
    "floor": "high",
    "roomType": "大床房"
  }' | jq
```

#### 2. 推荐结果示例
```json
{
  "success": true,
  "message": "为您推荐 9 个房间",
  "data": [
    {
      "id": 8,
      "room_number": "0801",
      "room_type": "标准间",
      "price": 259,
      "orientation": "south",
      "recommendScore": 155,
      "recommendReasons": [
        "价格合适",
        "高楼层",
        "南朝向",
        "高评分房间",
        "设施完善"
      ]
    },
    {
      "id": 5,
      "room_number": "0501",
      "room_type": "大床房",
      "price": 450,
      "recommendScore": 185,
      "recommendReasons": [
        "房型匹配",
        "价格优惠",
        "高楼层",
        "热门房间"
      ]
    }
  ]
}
```

#### 3. 前端交互流程
1. 用户选择入住/退房日期
2. 点击"🤖 智能推荐房间"按钮
3. (可选) 设置筛选条件: 房型/价格/楼层
4. 系统展示推荐房间卡片,显示:
   - 房间号 + 房型
   - 价格/晚
   - 推荐分数(满分100+)
   - 推荐理由标签
5. 点击选择推荐房间
6. 自动计算总金额 = 价格 × 天数

### 推荐算法详解

#### 评分维度
| 维度 | 权重 | 说明 |
|------|------|------|
| 房型匹配 | +30分 | 精确匹配客户需求 |
| 历史偏好 | +20分 | 符合客户历史订房习惯 |
| 价格合理 | +20分 | 在预算范围内(≤70%更优) |
| 楼层偏好 | +15分 | 符合高/低楼层要求 |
| 朝向偏好 | +15分 | 南向/东向等 |
| 房间评分 | +20分 | 历史客户评价 |
| 热门房间 | +10分 | 预订次数>10 |
| 新房激励 | +5分 | 新房推广 |
| 设施完善 | +10分 | 配套设施≥5项 |

#### 实际案例
**客户需求**: 2晚,500元预算,喜欢高楼层  
**系统推荐**: 801房(8楼标准间,259元/晚)  
**推荐理由**:
- 价格259元 < 500元预算 → +20分(价格合适)
- 8楼属于高楼层 → +15分(楼层偏好)
- 南朝向采光好 → +15分(朝向优)
- 历史评分4.5+ → +20分(高评分)
- 配套齐全(Wi-Fi/空调/洗衣机等) → +10分

**最终得分**: 155分 (基础100 + 额外55)

### 预期效果
- ✅ 推荐结果合理,符合用户期望
- ✅ 推荐理由清晰,增强信任
- ✅ 提升订单转化率

## 📥 功能3: Excel数据导出

### 演示场景
财务月底对账,需要导出4月份所有携程渠道订单和财务明细。

### 演示步骤

#### 1. 导出订单数据
```bash
# 导出所有订单
curl "http://localhost:3100/api/export/orders" -o all_orders.xlsx

# 导出4月订单
curl "http://localhost:3100/api/export/orders?startDate=2026-04-01&endDate=2026-04-30" \
  -o april_orders.xlsx

# 导出携程渠道订单
curl "http://localhost:3100/api/export/orders?channel=ctrip" \
  -o ctrip_orders.xlsx

# 组合筛选
curl "http://localhost:3100/api/export/orders?startDate=2026-04-01&endDate=2026-04-30&channel=ctrip" \
  -o ctrip_april.xlsx
```

#### 2. 导出财务数据
```bash
# 导出4月财务数据
curl "http://localhost:3100/api/export/finance?year=2026&month=4" \
  -o finance_202604.xlsx

# 导出3月财务数据(对比)
curl "http://localhost:3100/api/export/finance?year=2026&month=3" \
  -o finance_202603.xlsx
```

#### 3. Excel文件结构

##### 订单导出文件 (orders_YYYYMMDD_HHmmss.xlsx)
**工作表1: 订单列表**
| 订单号 | 渠道 | 客户姓名 | 联系电话 | 房间号 | 房型 | 入住日期 | 退房日期 | 订单金额 | 状态 | 创建时间 |
|--------|------|----------|----------|--------|------|----------|----------|----------|------|----------|
| ORD1713... | 携程 | 张三 | 138xxxx | 301 | 大床房 | 2026-04-20 | 2026-04-22 | 900 | 已确认 | 2026-04-18... |

**工作表2: 统计汇总**
| 总订单数 | 总营收 | 导出时间 |
|----------|--------|----------|
| 24 | 28794.73 | 2026-04-18 15:30:45 |

##### 财务导出文件 (finance_YYYYMMDD_HHmmss.xlsx)
**工作表1: 财务明细**
| 日期 | 类型 | 类别 | 金额 | 说明 | 创建时间 |
|------|------|------|------|------|----------|
| 2026-04-16 | 收入 | 房费 | 3445.03 | 订单 ORD... | 2026-04-16... |
| 2026-04-15 | 支出 | 水电费 | 230.50 | 3月份水电费 | ... |

**工作表2: 月度汇总**
| 年月 | 总收入 | 总支出 | 净利润 | 导出时间 |
|------|--------|--------|--------|----------|
| 2026-04 | 28794.73 | 5432.10 | 23362.63 | 2026-04-18... |

**工作表3: 渠道统计**
| 渠道 | 订单数 | 总营收 |
|------|--------|--------|
| 携程 | 8 | 11011.14 |
| 美团 | 6 | 6326.57 |
| 飞猪 | 5 | 4201.77 |

#### 4. 前端操作流程
**订单页面导出**:
1. 进入订单列表页
2. 点击"📥 导出"按钮
3. 弹出筛选弹窗:
   - 选择开始日期
   - 选择结束日期
   - 选择渠道(可选)
4. 点击"确认导出"
5. H5平台: 自动下载文件
6. 小程序: 复制下载链接

**财务页面导出**:
1. 进入财务统计页
2. 选择月份(如"2026-04")
3. 点击"📥 导出财务"按钮
4. 直接下载或复制链接

### 文件命名规范
```
orders_20260418_153045.xlsx      # 订单_年月日_时分秒
finance_20260418_153045.xlsx     # 财务_年月日_时分秒
```

- ✅ 中文内容显示正常
