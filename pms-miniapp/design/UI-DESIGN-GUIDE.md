## 🖼️ TabBar 图标设计

### 设计规范
- **尺寸:** 48x48 (设计稿) → 81x81px (导出PNG)
- **风格:** 简约线性，2.5px描边
- **状态:** 两种状态（未选中灰色 #7A7E83 / 选中蓝色 #4A90E2）

### 图标列表

#### 1. 首页图标 (home)
**设计概念:** 房子轮廓 + 门
- 未选中: 灰色线性图标
- 选中: 蓝色线性图标

#### 2. 消息图标 (message)  
**设计概念:** 气泡对话框 + 红色徽章
- 未选中: 灰色对话框
- 选中: 蓝色对话框
- **特殊:** 右上角显示99+红色徽章 (#FF4757)

#### 3. 房态图标 (calendar)
**设计概念:** 日历 + 日期点
- 未选中: 灰色日历
- 选中: 蓝色日历
- 包含日历网格和日期点装饰

#### 4. 统计图标 (finance)
**设计概念:** 柱状图(三根递增柱子)
- 未选中: 灰色柱状图
- 选中: 蓝色柱状图
- 三根柱子高度递增，形成上升趋势

#### 5. 我的图标 (mine)
**设计概念:** 人物头像 + 肩膀轮廓
- 未选中: 灰色人像
- 选中: 蓝色人像

### 文件路径
```
static/images/
├── home.svg              (首页-未选中)
├── home-active.svg       (首页-选中)
```

## 📱 首页设计

### 布局结构
```
┌─────────────────────────────┐
│  欢迎区 (渐变蓝色背景)        │
│  - 欢迎文字 + 日期            │
│  - 通知按钮(带徽章)           │
├─────────────────────────────┤
│  核心统计卡片                 │
│  - 今日营业额 (渐变卡片)      │
│  - 可售房间 + 在住订单        │
├─────────────────────────────┤
│  房态快览卡片                 │
│  - 待入住/已入住/待离店等     │
├─────────────────────────────┤
│  快捷操作网格                 │
│  - 3列布局，6个功能入口        │
└─────────────────────────────┘
```

### 核心特性
1. **顶部欢迎区:** 渐变蓝色背景，显示当前日期和通知
2. **营业额卡片:** 大号数字 + 增长趋势指示
3. **房态快览:** 彩色指示点 + 数字统计
4. **快捷操作:** 渐变图标背景 + 清晰标签

### 交互效果
- 卡片点击: `transform: scale(0.98)` + 阴影变化
- 图标按压: `transform: scale(0.9)`
- 平滑过渡: `transition: all 0.3s ease`

## ✨ 设计原则

1. **一致性:** 统一的颜色、圆角、间距系统
2. **简洁性:** 去除不必要的装饰，聚焦核心功能
3. **可识别性:** 清晰的视觉层次，重要信息突出
4. **响应性:** 流畅的交互反馈，良好的触摸体验
5. **现代感:** 渐变、阴影、圆角等现代设计元素
