## 🚀 快速开始

### 方法一: 使用自动部署脚本

```bash
cd /root/.openclaw/workspace/pms-miniapp
./design/deploy-ui.sh
```

### 方法二: 手动部署

#### 1. 备份原文件
```bash
cp pages/index/index.vue pages/index/index-old.vue
cp pages.json pages-old.json
cp App.vue App-old.vue
```

#### 2. 应用新文件
```bash
# 首页
cp pages/index/index-optimized.vue pages/index/index.vue

# pages.json 已经更新
# App.vue 已经更新
# common/theme.scss 已经创建
```

#### 3. 转换图标 (重要!)
```bash
# SVG已生成在 static/images/ 目录
# 需要转换为 PNG (81x81px)

# 方案A: 使用在线工具
# https://svgtopng.com/
# 上传SVG → 设置尺寸81x81 → 下载PNG

# 方案B: 使用命令行 (需安装 ImageMagick)
for file in static/images/*.svg; do
  convert -background none -density 300 "$file" -resize 81x81 "${file%.svg}.png"
done
```

#### 4. 在开发工具中预览
- 打开 HBuilderX 或微信开发者工具
- 运行到微信小程序
- 检查tabBar图标和首页显示
