## 📱 新增页面

### 1. 房态日历 (`pages/room-calendar/room-calendar.vue`)

**功能特性**：
- 月份导航（←  2026年4月  →）
- 横向滚动日历表
- 房间列表（左侧固定列）
- 状态颜色标识
- 图例说明

**交互**：
- 点击左右箭头切换月份
- 横向滑动查看更多日期
- 点击格子查看房间详情

### 2. 创建订单 (`pages/create-order/create-order.vue`)

**表单字段**：
- 房间（必填）- 下拉选择
- 客户姓名（必填）- 文本输入
- 联系电话（必填）- 数字输入 + 格式验证
- 预订渠道（必填）- 下拉选择
- 入住日期（必填）- 日期选择
- 退房日期（必填）- 日期选择
- 订单金额（必填）- 自动计算/手动输入
- 备注（选填）- 多行文本

**智能功能**：
- 自动计算金额 = 房价 × 天数
- 日期变更自动更新金额
- 手机号格式验证（11位）
- 退房日期必须晚于入住日期

### 3. 清洁管理 (`pages/cleaning/cleaning.vue`)

**筛选功能**：
- 日期筛选（日期选择器）
- 状态筛选（全部/待清洁/清洁中/已完成）

**任务卡片**：
- 房间号 + 时间
- 负责人
- 状态标签
- 备注信息
- 操作按钮（开始清洁/完成）

**状态流转**：
```
待清洁 → [开始清洁] → 清洁中 → [完成] → 已完成
```

## 🎯 使用指南

### 创建订单流程

1. 点击订单页面右下角 **+** 按钮
2. 选择房间（自动填充房价）
3. 填写客户信息（姓名、电话）
4. 选择预订渠道
5. 选择入住和退房日期（自动计算金额）
6. 检查信息无误后点击"创建订单"
7. 成功后自动返回订单列表

### 查看房态流程

1. 点击底部导航栏"房态"
2. 查看当月房态日历
3. 点击左右箭头切换月份
4. 横向滑动查看更多日期
5. 点击格子查看房间详情

### 清洁管理流程

1. 进入"我的"页面
2. 点击"清洁管理"
3. 选择日期查看当天任务
4. 筛选状态（全部/待清洁/清洁中/已完成）
5. 点击"开始清洁"更新任务状态
6. 完成后点击"完成"
