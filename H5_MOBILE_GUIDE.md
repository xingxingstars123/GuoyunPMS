## 📱 在手机上测试

### 步骤 1: 开放防火墙端口

```bash
# 如果使用 UFW
ufw allow 8080/tcp
ufw reload

# 如果使用 iptables
iptables -A INPUT -p tcp --dport 8080 -j ACCEPT

# 或在云服务器控制台的安全组中添加规则：
# 端口: 8080, 协议: TCP, 来源: 0.0.0.0/0
```

## 🐛 常见问题

### Q1: 无法访问 8080 端口？

**A**: 检查防火墙：
```bash
# 查看防火墙状态
ufw status
# 或
iptables -L -n | grep 8080

# 开放端口
ufw allow 8080/tcp
# 或在云服务器控制台添加安全组规则
```

### Q2: 页面显示"加载失败"？

**A**: 检查后端 API 服务：
```bash
# 查看后端状态
ps aux | grep "node server.js"

# 测试 API
curl http://localhost:3100/api/dashboard/stats

# 查看日志
cat /root/.openclaw/workspace/pms-prototype/backend/server.log
```

### Q3: 手机浏览器无法访问？

**A**: 确保：
1. 服务器防火墙已开放 8080 端口
2. 云服务器安全组已配置
3. 使用服务器的公网 IP（43.173.91.161）
4. 手机网络正常

### Q4: 想要修改页面内容？

**A**: 直接编辑 HTML 文件：
```bash
vi /root/.openclaw/workspace/pms-h5-mobile/index.html
```

保存后刷新浏览器即可看到效果（纯静态页面）。
