## 🚀 使用建议

### 1. 首次使用
```bash
# 1. 检查权限
cd ~/.openclaw/workspace/skills/feishu-collaboration
./scripts/check-permissions.sh --all

# 2. 创建测试文档
./scripts/create-doc.sh \
  --title "测试文档" \
  --content examples/example-content.md

# 3. 发送测试文件
./scripts/send-file.sh \
  --file examples/test-file.txt \
  --message "测试文件发送"
```

### 2. 集成到工作流
可以将脚本集成到：
- CI/CD 流水线（自动生成文档）
- 定时任务（日报、周报）
- 项目启动脚本（自动创建项目文档）
